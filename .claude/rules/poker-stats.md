# Poker Statistics Rules

Before implementing any poker statistic:
1. **Web search** how **PokerTracker4**, **Hold'em Manager (HM2/HM3)**, and **Hand2Note (H2N)** define and calculate it
2. Use their definitions as the **source of truth**

Always look up poker and poker analysis **terminology in PokerTracker4** before using terms in code, comments, or docs.
