## PGL Coverage & Priorities

Currently implemented: **VPIP, PFR, BB/100, BB/100 All-in Adjusted** (4 of ~60+ industry metrics).

High-priority candidates (common across all tools):

| Priority | Metric | Notes |
|---|---|---|
| 1 | 3Bet / Fold to 3Bet | Universally tracked, core profiling stat |
| 2 | AF / AFq | Core aggression metrics |
| 3 | Flop/Turn/River C-Bet | Most-used postflop stats |
| 4 | Fold to Flop/Turn/River C-Bet | Counterpart to C-Bet |
| 5 | WTSD / W$SD / WWSF | Showdown tendency metrics |
| 6 | Attempt to Steal / Fold to Steal | Blind play metrics |
| 7 | RFI | Raise first in by position |
